HTML Anatomy ==> <p>Hello World!</p>

An opening tag (<p>)
The content ("Hello World!" text)
A closing tag (</p>)
A tag and the content between it is called an HTML element.

The Body

Only content inside the opening and closing body tags can be displayed to the screen.

HTML Structure

HTML is organized as a collection of family tree relationships.

Headings

In HTML there are six different heading elements. From <h1> to <h6>.

Divs

<div> element is short for 'division' or a container that divides the page into sections.

Attributes

Attributes are content added to the opening tag of an element and can be used in several different ways from providing information to changing styling ["name=value"].

Displaying Text

To display text in HTML we can use a paragraph or a span.

Styling Text

<em> tag renders the text italic. <strong> tag renders the text bold.

Line Breaks

<br> tag is used to add a line break.

Unordered Lists

<ul> tag is to create a list of items in no particular order. The <li> or list item tag is used to describe an item in list.

Ordered Lists

Ordered Lists (<ol>) are like unordered lists, except that each list item is numbered.

Images

<img src="image-location.jpg" />

Image ALts

<img src="image-location.jpg" alt="Description of the image" />

Videos

<video src="myVideo.mp4" width="320" height="240" controls>Video not supported</video>Controls attribute instructs the browser to include basic video controls such as pausing and playing.

Structure

<!Doctype HTML>

It tells the browser what type of document to expect, along with what version of HTML is being used in the document

The <html> Tag

Anything between the opening and closing <html> tags will be interpreted as HTML code.

The Head

The <head> element contains the metadata for a web page. Metadata is information about the page that isn’t displayed directly on the web page. It goes above our <body> element.

Page Titles

A browser’s tab displays the title specified in the <title> tag. The <title> tag is always inside of the <head>.

Linking to Other Web Pages

<a href="https://www.wikipedia.org/"> This Is A Link To Wikipedia </a> target="_blank"You can add links to a web page by adding an anchor element <a> and including the text of the link in between the opening and closing tags. Setting the target attribute to blank opens the relevant webpage in new window.

Linking to Relative Page

<a href="./contact.html"> Contact </a>Anchor tag is used with a relative path to link from the current HTML file to the contact.html file in the same folder.

Linking At Will

HTML allows you to turn nearly any element into a link by wrapping that element with an anchor element. For example an image can be made into a link by wrapping <img/> tag inside an <a> tag [<a><img/></a>].

Linking to Same Page

<p id="paragraph"></p><a href="#paragraph"> Paragraph </a>In order to link to a target on the same page, we must give the target an id. The target link is a string containing the # character and the target element’s id.

Comments

Comments begin with <!-- and end with -->. Any characters in between will be ignored by your browser.

Tables

Create a Table

We use <table> element to create a table in HTML.

Table Rows

<tr> element is used to add row inside a table.

Table Data

In HTML, you can add data using the table data element: <td>.

Table Headings

To add titles to rows and columns, you can use the table heading element: <th>.

Table Borders

We can use border style property in CSS to add border to our table.

Spanning Columns

<td colspan="2">Out of Town</td>Data can span columns using the colspan attribute. The attribute accepts an integer (greater than or equal to 1) to denote the number of columns it spans across.

Spanning Rows

The rowspan attribute is used for data that spans multiple rows. It accepts an integer (greater than or equal to 1) to denote the number of rows it spans across.

Table Body

The <tbody> element should contain all of the table’s data, excluding the table headings.

Table Head

Table’s column headings is stored inside <thead> element.

Table Footer

The bottom part of a long table can also be sectioned off using the <tfoot> element.

Semantic HTML

Header and Nav

A <header> is a container usually for either navigational links or introductory content containing <h1> to <h6> headings.
A <nav> is used to define a block of navigation links such as menus and tables of contents. It can be used inside of the <header> element but can also be used on its own.

Main and Footer

The element <main> is used to encapsulate the dominant content within a webpage.
<footer> element contains information such as:
Contact information
Copyright informaton
Terms of Use

Article and Section

The <article> element holds content such as articles, blogs, comments, magazines, etc.
A website’s home page could be split into sections for the introduction, news items, and contact information using <section> element.

The Aside Element

<aside> element can be used to add information such as:
Bibliographies
Endnotes
Comments
Pull quotes
Editorial sidebars
Additional information

Figure and Figcaption

<figure><img src="overwatch.jpg"><figcaption>This picture shows characters from Overwatch.</figcaption></figure><figure> is an element used to encapsulate media such as an image, illustration, diagram, code snippet, etc.
<figcaption> is an element used to describe the media in the <figure> tag. Usually, <figcaption> will go inside <figure>.

Audio and Attributes

<audio><source src="iAmAnAudioFile.mp3"type="audio/mp3"></audio>The <audio> element is used to embed audio content into a document. <source> element is used to encapsulate our audio link.The audio type is scpecified by using type and named what kind of audio it is. <audio autoplay controls> Controls displays the audio controls into the browser such as play and mute.

Video and Embed
